ウィンドウ右上にシステムレイヤを表示するプラグインです。

主な仕様は以下の通りです。
・表示する文字を指定するだけであとは勝手に表示してくれます。
・システムレイヤは常に表示されます。トランジションしても消えません。
※KAG版ではスクロールトランジションを行うとトランジションされてしまいます。KAGEX版では大丈夫です。
・セーブ、ロードには未対応です。システムレイヤの状態は保存されません。

吉里吉里2 version 2.30 Rev.2/KAG 3 version 3.30 Rev.2で動作確認をしています。

バグを発見したらメールもしくはブログ(http://kasekey.blog101.fc2.com/)の方へ連絡してください。
