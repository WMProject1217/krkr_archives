SEコマンド

[se]タグでは効果音の再生、停止などができます。

省略表記

seから始まる不明なタグ名は[se]タグのname属性かつplay属性と見なされます。
例:
@se01 time=1000 loop
; @se name=se01 play=se01 time=1000 loop と見なされる

@se01 stop=1000
; @se name=se01 play=se01 stop=1000 と見なされる
; stop属性がplay属性より優先されるのでse01の再生を停止できます
[se]タグの不明な属性名はplay属性の属性値と見なされます。
例:
@se test time=1000
; @se play=test time=1000 と見なされる

効果音の再生

属性名属性値説明
play音声ファイル名指定されたファイルを再生開始します。
storage音声ファイル名play属性の別名。
startラベル名再生開始位置を指定します。ループチューナで設定したラベル名を指定します。
省略時はファイルの最初から再生されます。
looptrue/falsetrueを指定するとループ再生します。省略時はfalse。
time時間(ms)フェードイン時間を指定します。省略時はフェードせずすぐに再生開始します。
synctruetrueを指定すると再生が同期実行されます。つまり再生終了を待つようになります。
ただしループ再生の場合は再生終了しないため待てません。
nosynctruetrueを指定すると再生が非同期実行されます。つまり再生終了を待ちません。
デフォルトで非同期実行のためこの属性は常に無視されます。
wait時間(ms)指定すると再生が同期実行され、さらに終了してから追加で指定された時間だけ待ちます。
同期実行の詳細はsync属性の欄を参照してください。
canskiptrue/falsesync属性またはwait属性と一緒に使います。falseを指定すると終了待ちをクリックでスキップできなくなります。省略時はtrue。
bufバッファ番号再生に使うバッファを番号で指定します。
name音声ファイル名再生に使うバッファをファイル名で指定します。そのファイルを再生しているバッファで再生します。
再生しているバッファが存在しなければ未使用のバッファが自動的に使われます。
play属性にファイル名を指定して効果音を再生します。
使用するバッファが再生中なら現在のファイルを停止してから再生します。
buf属性とname属性をどちらも省略した場合は未使用のバッファが自動的に使われます。

効果音の停止

属性名属性値説明
stop時間(ms)再生中の効果音を停止します。
属性値を省略するとすぐに停止します。属性値に時間を指定するとその時間でフェードアウトして停止します。
time時間(ms)フェード時間を指定します。stop属性に数値を指定するのと同じです。
synctruetrueを指定するとフェードアウトが同期実行されます。つまりフェード終了を待つようになります。
nosynctruetrueを指定するとフェードアウトが非同期実行されます。つまりフェード終了を待ちません。デフォルトで非同期実行のためこの属性は常に無視されます。
wait時間(ms)指定するとフェードアウトが同期実行され、さらに終了してから追加で指定された時間だけ待ちます。
同期実行の詳細はsync属性の欄を参照してください。
canskiptrue/falsesync属性またはwait属性と一緒に使います。falseを指定するとフェード待ちをクリックでスキップできなくなります。省略時はtrue。
bufバッファ番号停止するバッファを番号で指定します。
name音声ファイル名停止するバッファをファイル名で指定します。そのファイルを再生しているバッファを停止します。
stop属性で再生中の効果音を停止します。
buf属性とname属性のどちらかは必ず指定してください。

効果音の音量設定

属性名属性値説明
fade音量%(0～100)効果音の音量を指定します。初期状態では100です。
time時間(ms)フェード時間を指定します。属性値を省略するとすぐに指定音量になります。
synctruetrueを指定するとフェードが同期実行されます。つまりフェード終了を待つようになります。
nosynctruetrueを指定するとフェードが非同期実行されます。つまりフェード終了を待ちません。デフォルトで非同期実行のためこの属性は常に無視されます。
wait時間(ms)指定するとフェードが同期実行され、さらに終了してから追加で指定された時間だけ待ちます。同期実行の詳細はsync属性の欄を参照してください。
canskiptrue/falsesync属性またはwait属性と一緒に使います。falseを指定するとフェード待ちをクリックでスキップできなくなります。省略時はtrue。
bufバッファ番号音量設定するバッファを番号で指定します。
name音声ファイル名音量設定するバッファをファイル名で指定します。そのファイルを再生しているバッファを設定します。
fade属性で効果音の音量を設定します。音量はplay属性で再生開始するたびに100にリセットされます。
buf属性とname属性のどちらかは必ず指定してください。
